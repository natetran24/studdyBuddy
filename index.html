How to Define a Table
A table is defined by < Table >
< tr >
Table Row; this starts a new row in the table
< th >
Table Heading; this creates a heading for the table
< td >
Table Data; this starts fills in the table's cells
< thead >
Groups the headings of the table
< tfoot >
Groups the footers of the table
< caption >
Adds a caption above the table
Difference between colspan and rowspan
Colspan combines cells between columns and rowspan combines cells between rows
Difference between cell spacing and cell padding
Cell spacing refers to the space between cells and cell padding refers to the width of the cells
